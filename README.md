# Data Science & Artificial Intelligence Bootcamp - Tuwaiq Academy

Welcome to my repository for the **Data Science & AI Bootcamp** at **Tuwaiq Academy**. This repository serves as a comprehensive portfolio containing all the hands-on tasks, labs, and real-world projects I completed during this intensive training program.

The bootcamp was delivered in Arabic and focused on moving from data engineering fundamentals to building advanced Machine Learning (ML) and Deep Learning (DL) architectures to solve practical, real-world problems.

---

## 🚀 Bootcamp Objectives & Core Focus
* **Data-Driven Solutions:** Transforming raw, complex datasets into actionable business insights.
* **Predictive Modeling:** Applying advanced ML and DL algorithms to build intelligent systems.
* **Advanced AI Fields:** Exploring practical applications in Computer Vision (CV) and Natural Language Processing (NLP).
* **Analytical Thinking:** Developing analytical problem-solving skills and professional project presentation techniques.

---

## 🛠️ Tech Stack & Tools
* **Programming Language:** Python
* **Data Analysis & Manipulation:** NumPy, Pandas
* **Machine Learning:** Scikit-learn
* **Deep Learning:** TensorFlow
* **Environment:** google colab / VS Code
